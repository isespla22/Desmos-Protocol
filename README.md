# DemosProtocol

A mathematical framework for distributed democratic consensus.

## What is it?
DemosProtocol is a technology-agnostic protocol that replaces 
the question-formation monopoly in democratic systems with a 
formally verified iterative convergence process.

## The problem it solves
In every existing democracy, someone else decides what citizens 
vote on. DemosProtocol makes the question itself the output of 
a participatory mathematical process — not a political one.

## Key properties
- Convergence proven via Banach Fixed-Point Theorem
- Dual-layer: citizens + expert panels
- Manipulation-resistant (formal s/n bound)
- Modular: municipal, national, supranational
- Technology-agnostic
